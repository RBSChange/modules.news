Comment créer une thématique afin de classer les actualités

Les actualités peuvent être classées par thématique.

Pour créer une thématique :

  • Se rendre dans le module Sites et pages > Rubrique « Actualités »
  • Cliquer sur dans la colonne gauche
  • Remplir les champs relatifs aux propriétés de la rubrique.
  • Une fois la rubrique créée, revenir sur le module « Actualités ».
  • Ouvrir le module Sites et pages dans l'arbre de ressources
  • Insérer la rubrique précédemment créée depuis l'arbre de ressources dans le dossier racine du module « Actualités » (colonne de gauche de l'applicatif).
